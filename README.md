**Python-based pipeline for automated survey data cleaning, quality control, and multilingual text processing.**

</p>


**Features**

* Fuzzy brand matching
* Entropy-based response validation
* Straightlining detection
* Duplicate open-text detection
* MinHash similarity checks
* Multilingual NLP preprocessing
* Automated Excel export

**Technologies & Libraries**

* Python
* Pandas
* NumPy
* RapidFuzz
* Datasketch
* XML parsing

**Purpose**

DVJ Insights is a marketing research company where researchers receive large volumes of ad-hoc and pre-test survey data on a weekly basis. One of the most time-consuming and challenging parts of the workflow is the data cleaning and preparation process before the data can be used for reporting and analysis. Since many researchers are not specialized in advanced programming or data analysis, this pipeline was developed to automate a large part of the survey data cleaning workflow.

The project focuses on improving efficiency, reducing repetitive manual tasks, and creating a more scalable and standardized process for preparing survey datasets for downstream reporting and consumer insight analysis.
